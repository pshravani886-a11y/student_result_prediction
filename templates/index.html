About This Project

This system predicts a student's final exam score using Machine Learning algorithms. It analyzes multiple factors like study hours, attendance, assignments, sleep hours, parental education, and extracurricular participation to estimate academic performance.

The goal of this project is to demonstrate how data science and machine learning can be used to support educational decision making and performance analysis.
